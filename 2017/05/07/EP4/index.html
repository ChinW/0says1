EP4: BAT实习与Hackathon夺冠之后，是什么让我选择了网易游戏？

5月 07, 2017 in podcast

王钦是中山大学大四的学生，即将在毕业后加入网易游戏。他除了在腾讯、阿里巴巴和网易游戏实习过之外，还是复旦创客马拉松的冠军，微软编程之美冠军，台湾⼤学黑客松英特尔企业奖亚军等等。在访谈中我们聊了：

  • 他的Hackathon冠军项目究竟是什么？在比赛中有没有制胜技巧？
  • 三段实习给他带了什么？为什么最终选择加入网易游戏
  • 他对自己在游戏行业有怎么样的未来规划？对VR游戏的未来又有怎样的前瞻？
